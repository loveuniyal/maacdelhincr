# MAAC Delhi NCR Website

This repository contains the source code for the **MAAC Delhi NCR** website (`maacdelhincr.com`).

## 🚀 Key Features

*   **Premium Landing Page**: A responsive page built with HTML, CSS, JavaScript (GSAP, Lenis smooth scrolling).
*   **Blogs System**: Standardized SEO-optimized static blogs hub located under `/blogs` featuring categories and starter posts.
*   **SEO Configurations**: Built-in `sitemap.xml` and `robots.txt` configuration for search engine indexation.
*   **Responsive Layout**: Optimized for both desktop and mobile viewports.

## 📂 Project Structure

*   `/assets/` — Website styles, scripts (GSAP/Lenis wrappers), and image assets.
*   `/blogs/` — The blog system category pages (`animation.html`, `vfx.html`, `gaming.html`, `design.html`) and articles.
*   `/blogs/posts/` — Individual blog articles (e.g. `impact-of-generative-ai-on-graphic-design.html`).
*   `index.html` — Homepage landing page.
*   `sitemap.xml` — Google Search Console indexing sitemap.
*   `robots.txt` — Web crawler permission file.
*   `README.md` — Project documentation.
